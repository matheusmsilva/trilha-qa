# Introdução a Algoritmos

## O que é um algoritmo?

Um algoritmo é uma sequência de passos utilizada para resolver um problema ou executar uma tarefa.

Em programação, algoritmos são instruções que dizem ao computador exatamente o que deve ser feito.

---

# Exemplos do dia a dia

Usamos algoritmos o tempo todo, mesmo fora da programação.

## Exemplo: fazer café

1. pegar a água
2. colocar água na cafeteira
3. adicionar café
4. ligar a cafeteira
5. esperar o café ficar pronto

Esses passos seguem uma ordem lógica para atingir um objetivo.

---

# Algoritmos na programação

Na programação, algoritmos ajudam a:

- validar informações
- tomar decisões
- repetir tarefas
- automatizar processos
- resolver problemas

---

# Exemplo simples em programação

## Problema

Verificar se um usuário pode acessar um sistema.

## Algoritmo

1. receber idade do usuário
2. verificar se idade é maior ou igual a 18
3. permitir acesso
4. caso contrário, bloquear acesso

---

# Características de um algoritmo

Um algoritmo deve ser:

- lógico
- organizado
- sequencial
- objetivo
- finito (precisa terminar)

---

# Exemplo em JavaScript

```javascript
let idade = 20;

if (idade >= 18) {
  console.log("Acesso permitido");
} else {
  console.log("Acesso negado");
}
```

# Por que algoritmos são importantes para QA?

Em automação de testes, praticamente tudo é construído através de algoritmos.

Exemplos:

- validar login
- preencher formulários
- comparar resultados
- repetir cenários
- verificar regras de negócio

Aprender algoritmos ajuda o QA a:

- desenvolver raciocínio lógico
- entender código
- criar automações
- investigar problemas com mais facilidade
